Rive Framer Player

Add a src parameter to load your Rive file

Example URL
player.html?src=https://your-url.com/file.riv
Available Parameters
src Rive file URL (required)
fit contain, cover, fill, fitWidth, fitHeight, none
align center, topLeft, topRight, bottomLeft, etc.
sm State machine name
artboard Artboard name
